## What is Corellia?
Corellia is a concept minimalist UI for a web browser, implemented with Visual Studio
## Installation :
== [DOWNLOAD](http://www.mediafire.com/file/lj7m7dm4l9qc4s1/Corellia.zip/file)==
1. run setup.exe / Corellia.application
2. Enjoy!
